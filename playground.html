Industry Recommendations

Adopt BEM-lite naming
Current naming mixes conventions (.dc-sim-lbl, .cmp-frag-card-name, .s-meta). Standardize on .block-element--modifier for new components. Existing code can migrate incrementally.
Extract section-label mixin
16 classes share identical declarations. A single .sec-label class eliminates ~96 duplicate lines of CSS. Use the existing .sec-label from layout.css as the canonical version.
Formalize the "dot" primitive
Already partially done (.picker-fdot, .s-fdot, .nf-dot share one rule). Add size modifiers: .dot--sm (8px), .dot--md (12px), .dot--lg (18px + abbr).
Use CSS layers for specificity
Modern CSS @layer would let tokens < components < layout < responsive cascade cleanly, eliminating the handful of !important overrides in the codebase.

Accessibility

Focus Ring
Global :focus-visible uses double box-shadow (paper gap + resin ring). Inputs override with border-color. Tab through the controls above to verify.
Touch Targets
--touch-target: 44px applied to all .tab elements via min-height. WCAG 2.5.5 compliant.
Reduced Motion
@media (prefers-reduced-motion: reduce) disables all animations and transitions globally.
Screen Reader
.sr-only utility for visually-hidden content. #cat-live for aria-live announcements.
